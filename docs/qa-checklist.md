# 质检清单

每批文档至少抽样检查以下项目。

## 1. 正文完整性
- 标题是否缺失
- 段落是否大量断裂
- 是否出现大段漏字
- 前言、目录、正文、附录是否被错误混合

## 2. 版式还原
- 多栏内容是否串行错乱
- 脚注是否错位
- 表格是否明显损坏
- 图片说明是否被丢失
- 公式、专名、引文是否异常

## 3. OCR 质量
- 人名、地名、组织名是否明显识别错误
- 年月日是否有高频错字
- 繁简、旧字形、异体字是否需要保留或归一
- 是否存在连续乱码

## 4. 元数据
- `doc_id` 是否唯一
- 标题、来源、日期、卷册是否齐全
- 是否标明扫描件
- 是否记录页码范围

## 5. 可用于后续 LLM 整理
- 标题层级是否可识别
- 章节边界是否清晰
- 段落长度是否适合切片
- 是否保留原始出处线索

## 6. 失败标签

出现问题时建议打标签：

- `ocr_bad`
- `layout_bad`
- `table_bad`
- `footnote_bad`
- `metadata_missing`
- `needs_manual_review`

## 7. 重点人工复核优先级

优先人工复核：
- 重要著作
- 关键历史时期材料
- 扫描质量差的 PDF
- 后续要用于网站展示或售卖的核心内容
