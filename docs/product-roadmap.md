# 产品路线图

## 总体目标

把毛泽东相关书籍与史料，从原始素材逐步沉淀为可检索、可关联、可展示、可产品化的知识资产。

## 工作流总览

1. 素材采集
2. PDF / 文档转 Markdown
3. Markdown 清洗与标准化
4. LLM 抽取实体、概念、关系、时间线
5. 构建知识档案与概念卡片
6. 建设知识图谱
7. 做成网站或知识产品

## 核心产物

### 1. 基础知识库文件
- clean markdown
- 结构化元数据
- 切片索引
- 来源回溯信息

### 2. 知识档案
- 人物档案
- 事件档案
- 组织档案
- 地点档案
- 著作档案

### 3. 概念层产品
- 概念卡片
- 术语解释
- 主题专题页
- 争议问题脉络

### 4. 网络化知识结构
- 实体关系图
- 时间线图谱
- 文献引用网络
- 主题聚类与交叉链接

## 阶段拆分

### Phase 1：书籍素材整理
- 用户手动搜集书籍
- 建立原始文件库
- 转成 Markdown
- 固化清洗与质检流程

### Phase 2：LLM 知识整理
- 概念识别
- 实体抽取
- 关系识别
- 内部链接
- 档案页生成

### Phase 3：知识图谱探索
- 参考 `graphify` 等现成项目
- 优先评估“是否可复用”而非从零重建
- 明确图谱 schema：实体、关系、来源、置信度

### Phase 4：网站与商业化
- 知识浏览网站
- 检索与探索界面
- 付费专题或会员访问
- 卡片、专题、人物档案、事件档案产品化
